WHAT IS IT ? 

  • FilmBox is a web application designed using the Model View Controller design pattern. This particular program uses MVC 5.

How does it work?

  • You register on the website and the customer is able to select movies he/she wishes to rent.  As there is 2 views to the website (one for the customer/one for the staff)  the staff have the ability to add, edit, sort and delete customers and movies.

Who is it for?

  • This application is best suited for a movie streaming website for example.

 

Which technologies or languages does this application use?

  • C#

  • ASP.NET

  • Entity Framework

  • CSS

  • HTML 5

  • Bootstrap

  • SQL
